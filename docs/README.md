# Grievance Redressal Backend Documentation

Welcome to the documentation for the Grievance Redressal Backend! This project provides a robust and scalable API for managing grievances, built with FastAPI, PostgreSQL, and JWT authentication.

## Highlights

*   **FastAPI:** Modern, fast (high-performance), web framework for building APIs with Python 3.8+.
*   **PostgreSQL:** Powerful, open-source object-relational database system.
*   **SQLAlchemy:** Python SQL toolkit and Object Relational Mapper (ORM) for efficient database interaction.
*   **JWT Authentication:** Secure, token-based authentication for API access.
*   **Alembic:** Lightweight database migration tool for SQLAlchemy.
*   **Modular Design:** Clear separation of concerns with dedicated modules for authentication, complaints, departments, and AI services.

For detailed information on setup, API endpoints, and specific functionalities, please refer to the dedicated documentation files within this directory.
